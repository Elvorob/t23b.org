# US-03: Join CTAs (Become a Boy Scout / Become a Cub Scout)

## User Story

**As a** parent or youth  
**I want** to see how to join as a Boy Scout or Cub Scout, with contact info and a "Learn more" option  
**So that** I can reach out or get more details.

## Acceptance Criteria

- [ ] The "Become a Boy Scout" block is visible.
- [ ] The "Become a Cub Scout" block is visible.
- [ ] Each block shows at least one contact name.
- [ ] Each block shows contact method(s): email and/or phone (or expected masked/obfuscated variant).
- [ ] A "Learn more" link (or equivalent) is present in each block.
- [ ] Each "Learn more" link navigates to the How to Join page (or intended destination).

## Notes

- Scope: https://www.t23b.org/
- Email may be obfuscated (e.g., via Cloudflare); validate presence, not raw address.

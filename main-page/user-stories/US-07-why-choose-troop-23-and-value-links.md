# US-07: Why Choose Troop 23 and Value Links

## User Story

**As a** visitor  
**I want** to see why Troop 23 is a good choice and follow links to Patrol Method, scoring, merit badges, Eagle Scouts, Procedures, events, and awards  
**So that** I can dig deeper into the troop's offerings.

## Acceptance Criteria

- [ ] The "Why Choose Troop 23?" section is visible.
- [ ] Value propositions (e.g., patrol leadership, advancement, merit badges) are present in text.
- [ ] A link to the Patrol Method / Patrols page is present and navigates correctly.
- [ ] A link to the Patrol Scoring page is present and navigates correctly.
- [ ] A link to Merit Badge Counselors is present and navigates correctly.
- [ ] A link to Eagle Scouts is present and navigates correctly.
- [ ] A link to the Procedures section is present and navigates correctly.
- [ ] A link to events/outdoor activities is present and navigates correctly.
- [ ] A link to the 50 Miler Award (or equivalent) is present and navigates correctly when referenced.

## Notes

- Scope: https://www.t23b.org/
- Some links may point to subsections; validate destination page or section.

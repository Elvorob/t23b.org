# US-06: What is Scouting and Patrol Links

## User Story

**As a** visitor  
**I want** to understand what scouting is and follow links to Patrols and Patrol Scoring  
**So that** I can learn about the program and structure.

## Acceptance Criteria

- [ ] The "What is Scouting?" section is visible.
- [ ] Explanatory content about scouting (e.g., Scout Oath, Scout Law, Patrol Method) is present.
- [ ] A link to the Patrols page is present and navigates correctly.
- [ ] A link to the Patrol Scoring section/page is present and navigates correctly.

## Notes

- Scope: https://www.t23b.org/
- Validates both content visibility and navigation.

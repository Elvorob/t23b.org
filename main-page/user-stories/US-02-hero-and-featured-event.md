# US-02: Hero and Featured Event (Next Big Thing)

## User Story

**As a** visitor  
**I want** to see the "Next Big Thing" featured event with date, time, and a way to read more  
**So that** I can learn about the next big activity.

## Acceptance Criteria

- [ ] The "Next Big Thing" section is visible on the homepage.
- [ ] The featured event name is displayed (e.g., Pinewood Derby).
- [ ] The event date is shown and is in an understandable format.
- [ ] The event time range is shown (e.g., "7:00 pm — 9:00 pm").
- [ ] A "Read more" link (or equivalent CTA) is present.
- [ ] The "Read more" link navigates to the correct event detail page.

## Notes

- Scope: https://www.t23b.org/
- Content may change; tests should validate structure and link behavior.

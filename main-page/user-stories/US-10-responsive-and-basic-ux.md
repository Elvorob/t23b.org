# US-10: Responsive and Basic UX

## User Story

**As a** visitor on mobile or desktop  
**I want** the homepage to display correctly and key actions (links, CTAs) to be usable  
**So that** I can use the site on my device.

## Acceptance Criteria

- [ ] On desktop viewport, the page layout is coherent and key sections are visible without horizontal scroll (unless by design).
- [ ] On mobile viewport, the page is readable and key sections are accessible (e.g., no critical content cut off).
- [ ] Primary links and "Read more" / "Learn more" CTAs are clickable (or tappable on touch devices).
- [ ] No overlay or element permanently blocks access to main content or CTAs.
- [ ] Text is readable (contrast and size) in both viewports.

## Notes

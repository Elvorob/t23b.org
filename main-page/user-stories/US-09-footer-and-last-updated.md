# US-09: Footer and Last Updated

## User Story

**As a** visitor  
**I want** to see when the page was last updated  
**So that** I have a sense of content freshness.

## Acceptance Criteria

- [ ] A "Last updated" (or equivalent) indicator is visible on the page.
- [ ] A date is displayed in a reasonable format (e.g., "May 07, 2025" or "MMM DD, YYYY").
- [ ] The date is readable and not broken or empty.

## Notes

- Scope: https://www.t23b.org/
- Date value may change; validation can check format rather than fixed value.

# US-08: New to Scouting CTA

## User Story

**As a** person new to scouting  
**I want** a clear invitation to join and links to How to Join, Advancements, and What is Scouting  
**So that** I can take the next step.

## Acceptance Criteria

- [ ] The "New to Scouting?" section (or equivalent) is visible.
- [ ] Invitation or call-to-action text to join or transfer is present.
- [ ] A link to the How to Join page is present and navigates correctly.
- [ ] A link to the Advancements page is present and navigates correctly.
- [ ] A link to the What is Scouting? page is present and navigates correctly.

## Notes

- Scope: https://www.t23b.org/
- Ensures onboarding path is discoverable and functional.

# US-04: What's Due Soon

## User Story

**As a** scout or parent  
**I want** to see what is due soon (e.g., deposit, Summer Camp) with a deadline and link  
**So that** I do not miss important deadlines.

## Acceptance Criteria

- [ ] The "What's Due Soon" section is visible on the homepage.
- [ ] At least one due-soon item is displayed (e.g., deposit fee, event name).
- [ ] A deadline or date is shown (e.g., "Feb 28 tomorrow" or equivalent).
- [ ] Each item has a link to the related event or page.
- [ ] The link navigates to the correct destination (e.g., Summer Camp event page).

## Notes

- Scope: https://www.t23b.org/
- Content is time-sensitive; tests should allow for dynamic dates.

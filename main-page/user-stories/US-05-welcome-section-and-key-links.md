# US-05: Welcome Section and Key Links

## User Story

**As a** visitor  
**I want** to read about Troop 23 and use links to History, Trips, community service, location, and About Us  
**So that** I can explore further.

## Acceptance Criteria

- [ ] The "Welcome to Troop 23" section is visible.
- [ ] Introductory text about Troop 23 is present (e.g., since 1910, Bay Ridge, diversity, service).
- [ ] A link to the History page is present and navigates correctly.
- [ ] A link to the Trips/events page is present and navigates correctly.
- [ ] A link to the community service section or events is present and navigates correctly.
- [ ] A link to the meeting location (e.g., Our Lady of Angels Church) is present and navigates correctly.
- [ ] A link to the About Us page is present and navigates correctly.

## Notes

- Scope: https://www.t23b.org/
- Link targets may be full URLs or anchors; validate destination page/section.
